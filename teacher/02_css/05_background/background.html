CSS background

background-color

배경색을 지정한다.

background-clip

배경이 지정되는 범위를 설정한다.

  • border-box(기본값) : border 영역까지 지정된다.
  • padding-box : padding 영역까지 지정된다.
  • content-box : content 영역까지 지정된다.

background-image

이미지를 배경으로 지정한다.
url("파일경로") 형식의 값을 사용한다.

background-repeat

배경 이미지의 반복 방법을 지정한다.

  • repeat(기본값) : 가로, 세로방향으로 이미지를 반복한다.
  • repeat-x : 가로(x축)방향으로 이미지를 반복한다.
  • repeat-y : 세로(y축)방향으로 이미지를 반복한다.
  • no-repeat: 반복하지 않는다.

background-position

배경 이미지의 위치를 지정한다.
x축, y축 순으로 지정한다.
한가지만 지정 시 나머지는 center로 간주한다.

background-size

배경 이미지의 크기를 지정한다.
x축, y축 순으로 지정한다.
한쪽만 지정시 원본 비율에 자동으로 맞춘다.

  • auto : 원본 이미지의 크기만큼 표시한다.
  • contain : 이미지가 잘리지 않는 범위 내에서 가장 크게 표시한다.
  • cover : 이미지를 자르면서 요소를 가득 채우게 표시한다.

background-origin

배경 이미지가 시작되는 범위를 지정한다.

background-attachment

배경 이미지의 첨부 방식을 지정한다.

  • scroll(기본값) : 스크롤을 해도 배경 이미지가 고정된다.
  • local : 스크롤에 따라 배경 이미지가 함께 움직인다.
  • fixed : 뷰포트를 기준으로 고정된다. 배경 이미지로 지정된 요소를 통해서만 보인다.

background

배경관련 속성을 한번에 지정하는 단축속성이다.
다른 것은 순서가 상관없지만, position과 size는 /로 구분해서 작성해야한다.(position 먼저)
단축속성에 size를 포함시키고 싶으면 무조건 앞에 position을 지정해야한다.
