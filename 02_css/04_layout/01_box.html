블록 레벨 요소

블록레벨요소는 영역을 지정할 때 사용한다.

블록 요소는 너비와 높이가 지정이 가능하다.

블록 요소의 너비는 기본적으로 부모의 너비 내에서 최대 너비를 가진다. (기본값 auto)

블록 요소의 높이는 기본적으로 자신이 가진 컨텐츠만큼 늘어난다.

너비와 상관없이 줄바꿈을 일으킨다

인라인 레벨 요소

특정 컨텐츠를 지정할 때 사용한다

인라인 요소는 너비와 높이 지정이 불가능하다.

인라인 요소는 자신이 가진 컨텐츠만큼의 너비와 높이를 가진다.

width/height

% : 부모 너비/높이에 대한 백분위로 지정됨
