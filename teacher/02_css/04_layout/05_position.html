CSS position

position

요소의 배치 방법을 지정한다.

  • static(기본값) : 문서의 흐름대로 나열된다. 오프셋 지정이 안된다.
  • relative : 기본적으로 static처럼 문서의 흐름대로 나열된다. 오프셋의 기준이 문서 흐름상 자신의 원래 위치가 된다.
  • absolute : 자신의 기준이 되는 상위 요소 기준으로 움직인다.
    상위 요소들 중 position 값이 static이 아닌 요소가 기준이 된다. inline-block과 비슷하게 동작한다.
    float 처럼 다른 요소의 위나 아래에 위치한다.
  • fixed : 뷰포트가 기준이 된다.

offset(top, bottom , left, right)

position 값에 따른 기준이 되는 요소로부터 위치를 잡는다.

z-index

absolute나 fixed와 같은 요소 포개어졌을 때 더 위에 보여줄 요소를 지정할 수 있다.
z-index가 더 높은 요소가 위로 올라온다.
