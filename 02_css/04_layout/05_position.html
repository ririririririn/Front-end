CSS position

position

요소의 배치 방법을 지정한다

  • static :
  • relative : 기본적으로 static처럼 문서의 흐름대로 나열된다. 오프셋의 기준이 문서 흐름상 자신의 원래 위치가 된다
  • absolute : 자신의 기준이 되는 상위 요소 기준으로 움직인다
    상위 요소들 중 position값이 static이 아닌 요소가 기준이 된다.
    inline-block과 비슷하게 동작한다
    float 처럼 다른 요소의 위 나 아래에 위치한다
    오프셋 값이 없으면 자신의 원래 위치에 자리한다 (top, right ... )
    z-index로 어느 것이 위로 아래로 올지 가능
  • fixed : 뷰포트가 기준이 된다.

offset(top,bottom,left,right)

position 값에 따른 기준이 되는 요소로부터 위치를 잡는다.

z-index

absolute나 fixed 같이 포개진 요소들 중 더 위에 보여줄 요소를 지정할 ㅅㅜ있다
더 높은 요소가 위로 올라옴
