CSS transition

요소의 CSS 속성값에 변화가 있을 때 변화되는 시간, 폭, 지연시간 등을 지정할 수 있다.

  • transition-property : 트래지션을 적용할 속성을 선택한다. 기본값은 all.
  • transition-duration : 트랜지션의 지속시간을 지정한다.
  • transition-delay : 트래지션의 지연시간을 지정한다.
  • transition-timing-function: 트래지션의 변화폭을 지정한다.
  • transition : 단축 속성. 순서는 duration과 delay 순서만 지키면 된다.
